India - State Boundaries

The State Boundaries of India, Scraped from ECI's Poling Station Locations Website.

  1. Some states have pre-delimitation boundaries.
  2. Some boundary data may be shifted.
  3. Some State Boundary names are incorrect or missing.
  4. Telangana still marked as Andhra Pradesh.
  5. UTs like Andaman & Nicobar, Chandigarh, etc. do not have state assemblies.

Shapefile Attributes

  • ST_NM: Name of the States as per the State Boundaries.
